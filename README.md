# hw-manager

Flask-based frontend to view Canvas assignments.
[Try here](stingray-app-ayu4b.ondigitalocean.app/)

## Stored Cookies
- `token`: Canvas API token
- `url`: Canvas page url
- `todo`: array of todo list items
- `removed-classes`: list of course ids to hide
